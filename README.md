# Adi — CTF Journey

Personal CTF writeup documentation. One file per challenge.

## Structure

```
CTF-REPO/
├── _Templates/
│   └── writeup-template.md     ← copy this for every new challenge
├── PicoCTF/
│   └── challenge-name.md
├── HackTheBox/
│   ├── Machines/
│   │   └── machine-name.md
│   └── Challenges/
│       └── challenge-name.md
├── TryHackMe/
│   └── room-name.md
└── Competitions/
    └── EventName-YYYY-MM/
        └── challenge-name.md
```

## Workflow

1. Copy `_Templates/writeup-template.md` into the right platform folder
2. Rename it to the challenge name in kebab-case (e.g. `format-string-0.md`)
3. Fill it out as you solve — document dead ends, not just the solution
4. Commit when done
